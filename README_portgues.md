Este projeto é focado em construir uma base para cadastro de disciplinas, notas e medias de um aluno, mas especificamente do SENAI Cimatec.
Porem, sua lógica é toda comentada, e 100% maleável para adaptação de outras instituições ou funcões. 
Atualmente ele conta com as funções:

1- Incluir disciplinas e suas notas;
2- Listar disciplinas pelo nome;
3- Alterar notas de uma disciplina;
4- excluir disciplina e suas notas;
5- Lançar nota final de uma disciplina(especifico para o cimatec);
6- Listar resumo academico;
7- Exibir array de médias;
8- Mostrar ranking de disciplinas;

O programa consta tambem uma função de armazenamento, então pode guardar arquivos no computador, sem precisar ficar aberto 100% do tempo.
