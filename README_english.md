This project focuses on building a foundation for managing student subjects, grades, and averages, specifically tailored for SENAI Cimatec.
However, the logic is fully commented and 100% adaptable for other institutions or purposes. 
Currently, it includes the following features:

1- Add subjects and their respective grades;
2- List subjects by name;
3- Edit grades for a specific subject;
4- Delete a subject and its grades;
5- Register final grades (specific to Cimatec's system);
6- List academic summary;
7- Display the averages array;
8- Show subject ranking;

The program also features a storage function, allowing it to save files to your computer so the application doesn't need to stay open all 
the time.
